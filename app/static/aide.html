Retour à l'application

Prérequis enquêteur

Compte ngrok : un compte ngrok vérifié est nécessaire pour générer le lien distant du participant.

Compte ou instance LiveKit : il faut une URL LiveKit, une API Key et une API Secret valides pour ouvrir la salle de visioconférence.

FFmpeg : FFmpeg est nécessaire pour produire "video.mp4" et "audio.mp3" après l'entretien.

Transcription Whisper : si vous voulez la transcription automatique, installez aussi `faster-whisper`. Ensuite, dans l'application, vous pouvez choisir la taille du modèle : `small` ou `medium`.

À quoi sert ngrok ?

ngrok crée une URL publique en "https" vers l'application qui tourne sur l'ordinateur de l'enquêteur (votre ordinateur). Dans cette application, c'est la solution utilisée pour donner accès au participant. Le participant n'installe rien : il ouvre simplement le lien que vous allez lui envoyer.

De quoi le participant a besoin ?

Aucune librairie à installer

Le participant n'installe ni Python, ni LiveKit, ni ngrok. Tout passe par le navigateur.

Un navigateur récent

Chrome, Edge, Firefox ou Safari récent suffisent, avec caméra et micro autorisés.

Une connexion internet

Il lui faut seulement le lien envoyé par l'enquêteur, une connexion internet stable et un ordinateur avec webcam.

Où renseigner les clés LiveKit ?

La capture ci-dessous montre les informations LiveKit à récupérer. Dans cette application, vous devez ensuite les saisir dans "2. Configuration LiveKit" : "URL LiveKit", "API Key" et "API Secret".

Stockage local

Une fois enregistrées, ces informations sont stockées dans le fichier "config.json" à la racine du projet.

Installation de FFmpeg

"FFmpeg" est utilisé pour produire "video.mp4" et "audio.mp3". Sans lui, les exports et une partie du traitement ne fonctionneront pas.

Librairies à installer sur Mac

Fichier de transcription

Le fichier "requirements.txt" regroupe les dépendances Python nécessaires à la transcription.

Librairies Python pour la transcription

Ce fichier installe surtout "faster-whisper" et ses dépendances Python, notamment "numpy<2".

python3 -m venv .venv source .venv/bin/activate pip install -r requirements.txt

FFmpeg

"ffmpeg" est nécessaire pour générer "video.mp4" et "audio.mp3".

brew install ffmpeg

ngrok

"ngrok" est nécessaire pour créer le lien distant du participant.

brew install ngrok
